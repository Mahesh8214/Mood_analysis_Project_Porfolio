Project Execution Guide

Introduction

Your project aims to provide a seamless experience for students and parents through a Flutter-based application and a web-based dashboard.

Project Setup

To get started, follow these steps:

  1. Install Flutter SDK verison >=3.2.3 <4.0.0 .
  2. Clone the student application repository.

    3. Git Repository

    https://github.com/yourusername/yourrepository

  3. Configure Firebase for authentication and database in student side application.
  4. Access the parent web page repository.
  5. Configure Firebase for authentication and database.

Student Side Application

The student side application allows users to:

  • Log in with their credentials.
  • Use the keyboard for sentence input.
  • Automatically detect and send sentences to the Firebase Realtime Database.

Parent Side Web Page

The parent side web page provides:

  • Setup python framework flask in your enviroment
  • Install all the requirements file by running below code

    • pip install -r requirements.txt

  • Setup the firebase credential same as student side
  • Add the firebase key for retriving data
  • Then run the app.py file
  • It will show you the login then there is analysis of the mood student

Additional Features

In addition to the core functionalities, the project also includes:

  • Real-time synchronization between student and parent sides.
  • Customizable user profiles.
