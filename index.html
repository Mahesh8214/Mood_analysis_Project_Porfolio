Motive of our project


Project Goal:

The primary goal of Sentiment Analysis is to proactively monitor and address the emotional well-being of students. By analyzing their sentiments and emotions, we aim to intervene early to prevent depression, mental health disturbances, and suicidal tendencies. Our project focuses on providing real-time insights into students' moods to empower parents and educators to offer timely support and guidance.

Student Application (Flutter):


Developed using Flutter, our student application enables users to input their thoughts and feelings.


These inputs are securely stored in a database for further analysis.


Parent Dashboard (Flask Web Application):


The parent dashboard, built using Flask, allows parents to monitor their child's mood in real-time.


If the system detects concerning emotions like depression or suicidal thoughts, it sends immediate notifications to parents, alerting them to take appropriate action.
